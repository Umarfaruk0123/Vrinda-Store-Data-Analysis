# 🛍️ Vrinda Store Annual Sales Data Analysis — Excel Project

A complete sales data analysis project for **Vrinda Store (2022)** built using **Microsoft Excel**. The project involves cleaning raw data, creating pivot tables, and building an interactive dashboard to uncover key business insights.

## 📊 Dataset Overview

The raw dataset contains **31,047 records** with **21 columns** including:

| Column | Description |
|--------|-------------|
| Order ID | Unique order identifier |
| Cust ID | Customer identifier |
| Gender | Customer gender (Men/Women) |
| Age / Age Group | Customer age and age bracket |
| Date / Month | Order date and month |
| Status | Order status (Delivered, Cancelled, etc.) |
| Channel | Sales platform (Amazon, Flipkart, etc.) |
| Amount | Order value |
| Ship State / Country | Delivery location |
| B2B | Business-to-business flag |

---

## 📈 Analysis Performed

The project includes **6 analysis sheets** with Pivot Tables and Charts:

| Sheet | Analysis |
|-------|----------|
| **Order vs Sales** | Monthly order count vs total sales amount |
| **Men vs Women** | Sales split between male and female customers |
| **Order Status** | Breakdown of Delivered, Cancelled, Returned & Refunded orders |
| **Top 5 States** | Top 5 states by total sales amount |
| **Age and Gender** | Order distribution across age groups and gender |
| **Channels** | Sales performance across different platforms |

---

## 🔍 Key Insights

- 👩 **Women** contribute significantly more to total sales than men
- 🏆 **Maharashtra** leads among all states in total revenue
- 📦 Majority of orders have a **Delivered** status
- 🛒 **Amazon** is the highest performing sales channel
- 👧 The **Adult** age group (30–49 yrs) places the most orders
